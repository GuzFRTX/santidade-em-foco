# Intro “GUZ” Size Design

## Goal

Improve readability of the “GUZ” brand credit on the initial loading screen.

## Design

Apply `font-size: 1.06rem` only to `.experience-intro__message small span`, which wraps “GUZ”. This is approximately 10% larger than the current `0.96rem`: prominent enough to stay legible while remaining visually secondary to the main message and start button. Preserve the surrounding credit text, spacing, typography, animation, and responsive layout.

## Verification

Confirm the selector still targets only “GUZ” and run the project’s relevant automated checks.
