# Referência visual e estrutural

Site de inspiração:
https://landonorris.com/

## O que usar como inspiração

Quero usar o site como referência de experiência, ritmo visual e estrutura narrativa, não como cópia literal.

## Ideias principais da referência

- Página inicial com experiência visual forte
- Scroll vertical imersivo
- Uso de imagens grandes
- Transições suaves entre seções
- Sensação editorial/premium
- Pouco texto por seção
- Ritmo visual cinematográfico
- Página inicial como narrativa, não como landing page comum

## O que NÃO copiar

- Não copiar identidade visual da Fórmula 1
- Não copiar textos
- Não copiar imagens
- Não copiar branding
- Não copiar o rosto/personagem interativo
- Não copiar layout pixel a pixel

## Adaptação para este projeto

Este projeto é um portfólio para fotógrafo/videomaker.

No lugar do rosto interativo inicial, a home deve começar com o nome do fotógrafo usando o componente Video Text da Inspira UI, com filmagens preenchendo o texto.

## Estrutura desejada da Home

1. Hero com Video Text
2. Scroll narrativo com fotos selecionadas
3. Seção de manifesto visual
4. Linha do tempo
5. Chamada para página de projetos

## Estrutura desejada da página Projetos

1. Hero da página
2. Filtros por categoria
3. Filtros por evento
4. Galeria de fotos
5. Seção de vídeos no final
6. CTA de contato
